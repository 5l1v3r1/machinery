    Machinery

    Machinery is a State Machine library for structs in general. It also aims to have (when implemented with Phoenix) an optional build-in GUI that will represent each resource’s state.

    Installation

    If available in Hex, the package can be installed by adding machinery to your list of dependencies in mix.exs:

    def deps do
  [
    {:machinery, "~> 0.1.0"}
  ]
end
